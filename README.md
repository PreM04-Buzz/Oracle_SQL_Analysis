# Project 02 – Analysis for Data using Oracle SQL

This project implements an end-to-end data analysis pipeline using:
• Oracle Autonomous Database
• Flask front-end web application
• Staging → ETL → Normalized Schema workflow
• CSV/XLSX file upload
• Data preview with Pandas
• Interactive Charts (Bar, Line, Scatter) using Chart.js

## Features
- Upload CSV/XLSX files from a web interface
- Data stored in Oracle DB staging table
- PL/SQL ETL cleans, standardizes, and loads dimensions + fact table
- Preview top rows before analysis
- Dynamic chart generation
- Command box for custom operations

## Technologies Used
- Python (Flask, Pandas)
- Oracle Database 23c / ATP Cloud Wallet
- HTML + CSS + Chart.js
- SQL + PL/SQL
- Git/GitHub for version control
